## Installation

You'll need to have [Node.js] (https://nodejs.org/) to get started.

```bash
$ npm install # Install dependencies
```

* If your having issues try updating to the latest version of node.


#### DEV Build
```bash
$ npm run build:dll
$ npm run start [-h 10.1.4.666] [-p 1488]
```

#### PROD Build
```bash
$ npm run build:prod
```
